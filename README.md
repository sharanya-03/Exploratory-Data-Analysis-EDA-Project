# Exploratory Data Analysis (EDA) - Sample Superstore Dataset

## Project Description

This project performs Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python.

The goal is to understand the data, identify trends and patterns, analyze sales and profit, and gain business insights through visualizations.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features

The dataset contains information about:

- Orders
- Customers
- Products
- Sales
- Profit
- Discounts
- Regions

## Analysis Performed

- Data Cleaning
- Missing Value Check
- Statistical Summary
- Sales Analysis
- Profit Analysis
- Correlation Analysis
- Data Visualization

## Visualizations

The following charts were created:

- Sales Distribution Histogram
- Profit Distribution Histogram
- Box Plots
- Category-wise Sales Chart
- Region-wise Profit Chart
- Sales vs Profit Scatter Plot
- Correlation Heatmap

## Key Insights

- Technology category generated high sales.
- Discounts can reduce profits.
- Sales and profit are positively related.
- Some products generate high sales but low profit.
- Regional performance varies across the business.
